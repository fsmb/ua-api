## PartialDate

Partial date.

| Name | Type | Required | Description |
| - | - | - | - |
| Month | string (mm) | No | Two digit month. |
| Day | string (dd) | No | Two digit day. |
| Year | string (yyyy) | No | Four digit year. |
| Date | string (date) | No | Date if all date parts are available. |
